# Lotto Number Generator

A simple web-based Lotto number generator with dark mode and a partnership inquiry form.

## Features
- **Lotto Generation**: Generates 5 sets of random lottery numbers (1-45) at once.
- **Theme Toggle**: Support for Dark Mode and White Mode (persisted in localStorage).
- **Inquiry Form**: Partnership inquiry form powered by Formspree.
- **Responsive Design**: Works on mobile and desktop.

## How to use
1. Open `index.html` in any web browser.
2. Click "Generate Numbers" to get 5 sets of lottery numbers.
3. Use the top-right button to toggle between Dark and White modes.
4. Fill out the contact form at the bottom for partnership inquiries.

## Tech Stack
- HTML5
- CSS3 (Vanilla)
- JavaScript (ES6+)
- [Formspree](https://formspree.io/) for form handling.
